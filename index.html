This non-profit project was created by Udo Wiegärtner with the help of Claude Code.

The project was inspired by the great book "Thing Explainer" by Randall Munroe.

The English version uses mainly the list of the ~1000 most common English words assembled and shared by Randall Munroe.

For more information about the English and German word lists see the descriptions on GitHub.
